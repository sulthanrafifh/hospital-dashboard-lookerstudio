# hospital-dashboard-lookerstudio
Hospital data monitoring dashboard built using Google Looker Studio to analyze patient statistics, recovery rates, and room availability.
## Dashboard Link
https://lookerstudio.google.com/reporting/aba55622-b0af-4afb-a1c4-1520878f61b7

## Features

* **Patient Monitoring Overview**
  Displays total number of patients and real-time distribution of patients across hospital rooms (Orchid, Lily, and Lavender).

* **Room Occupancy Tracking**
  Visualizes how many patients are currently assigned to each hospital room.

* **Patient Records Table**
  Provides a detailed list of patient information including room, disease, doctor in charge, admission date, discharge date, and status.

* **Disease Distribution Analysis**
  Treemap visualization showing the proportion of patients by disease category.

* **Patient Admission vs Discharge Trend**
  Time series chart comparing the number of patients admitted and discharged over time.

* **Recovery and Mortality Metrics**
  Key performance indicators including number of recovered patients, number of deaths, recovery rate, and mortality rate.

* **Average Length of Stay (ALOS)**
  Calculates the average hospitalization duration of patients.

* **Average Length of Stay by Disease**
  Compares treatment duration across different disease categories to identify cases requiring longer care.
